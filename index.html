Analytics Projects

A collection of data analysis work — from EDA to predictive modeling

Customer Churn Prediction

Built a logistic regression and random forest pipeline to flag at-risk customers. Surfaced top retention drivers via SHAP.

Engineered ~40 features from billing and engagement data, tuned with cross-validation, and exposed the scoring service behind a simple Flask API.

  • Python
  • scikit-learn
  • Pandas

Sales Trend EDA

Deep-dive exploration of three years of retail sales — seasonality, region splits, and category-level trend breaks.

Used decomposition, rolling stats, and statistical tests to separate signal from noise, with a final report shared with the business team.

  • Python
  • Pandas
  • Matplotlib

SQL Revenue Dashboard

Modeled a star schema and shipped a daily revenue dashboard covering MRR, cohort retention, and channel attribution.

Stack: Postgres + dbt + a thin BI layer. Includes data contracts, tests, and a documented metrics layer.

  • SQL
  • dbt
  • Postgres

NLP Sentiment Analysis

Classified product review sentiment using a fine-tuned transformer, with topic clustering for the negative cohort.

Compared a baseline TF-IDF + logistic model against a distilled BERT, with attention to data leakage and class imbalance.

  • Python
  • NLP
  • Transformers
