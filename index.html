Drug Data Analysis

Analyzed FDA drug approval data using Python and Amazon S3, uncovering trends in regulatory designations, patient reach, and manufacturer activity.

Loaded raw FDA records from S3, cleaned and modeled them in Python with Pandas, and produced an analytical write-up highlighting the most active regulatory pathways and the manufacturers driving them.

  • Python
  • Pandas
  • AWS S3

GLP-1 Analysis

End-to-end Python analysis of GLP-1 weight-loss drugs (Ozempic, Wegovy, Mounjaro, and related agents) using a public Kaggle master dataset — from adverse events and trials to market signals and anomaly detection.

Structured the work around seven analytical questions answered in Jupyter notebooks: FAERS hospitalization patterns by drug, stock price vs. search interest, geographic reporting, age and sex breakdowns, clinical-trial phase and sponsor mix, pipeline compounds, and monthly FAERS z-score anomalies. Built reusable loaders and cleaning utilities in Python, exported figures and summary tables, and summarized findings in report.md.

  • Python
  • Pandas
  • Jupyter
  • Kaggle

California Hospitals Dashboard

Risk-adjusted inpatient mortality and quality ratings across 528 California hospitals, surfaced through dynamic Tableau dashboards.

Built SQL and ETL workflows to model the dataset, then shipped interactive Tableau views highlighting performance, regional trends, and procedure-level outcomes to drive actionable insights for healthcare decision-makers.

  • Tableau
  • SQL
  • ETL

Nurse Stress Prediction

Demography Hiring Analysis

Replaced fragmented Excel workforce reporting with an automated Python pipeline over real Bureau of Labor Statistics data — surfacing how employment, unemployment, and labor-force participation differ by gender and age.

Cleaned and merged seven BLS Current Population Survey series into a single monthly analysis table, engineered employment, participation, and unemployment gaps, and produced decision-ready charts. Key insight: workforce differences in this window are driven more by labor-force participation than by unemployment, shifting the question from “who is hiring?” to “who is participating?”

  • Python
  • Pandas
  • Matplotlib

SAP Automation

A Python script that automates repetitive data extraction tasks in SAP, eliminating manual navigation and exports through SAP's built-in scripting interface.

Built on top of SAP's built-in Script Recorder: record a manual SAP GUI session, port the generated VBScript to Python with pywin32, and replay the steps automatically. The result is a reusable workflow that pulls reports and writes data back to SAP without manual clicks, saving hours of repetitive effort each week.

  • Python
  • SAP GUI
  • pywin32
